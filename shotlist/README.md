# shotlist/ — pre-visit filming shot lists

Auto-generated by `scripts/gen-shotlist.mjs` (command: `/shotlist`). Phase ① of
the site-visit video workstream — see [`../NAC-SITE-VISIT-VIDEO.md`](../NAC-SITE-VISIT-VIDEO.md).

Each listing produces:

- `<slug>.md` — markdown script (paste into Notion / print).
- `<slug>.html` — phone-friendly tickable checklist. Open on your phone on-site:
  `https://rayvtt.github.io/Nac-Property-Hub-Listing-PDP/shotlist/<slug>.html`
  (VI/EN toggle, progress ring, ticks saved on the device).
- `index.html` — trip pack when several listings are generated together.

Do not hand-edit — regenerate with `/shotlist <slug>`. Filming convention: shoot
each shot as `<slug>__<ID>.mp4` so the upload step can auto-route the clips.
